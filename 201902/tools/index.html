写一些文章的时候总会忘记部分 Markdown 的语法，或者是 LaTeX\LaTeXLATE​X 的符号，在网上搜索半天。放几个和博客配置比较切合的文档页面：

  • https://katex.org/docs/supported.html
    • 博客在用 KaTeX\KaTeXKATE​X 进行数学公式渲染，基于hexo-math的组件有小部分修改。
  • https://shd101wyy.github.io/markdown-preview-enhanced/#/zh-cn/markdown-basics
    • 逛 VSCode 下的预览组件，发现还有一个比较全面的帮助文档

我在 VSCode 下用的插件：

  • markdownlint 检查语言规范性，确保大部分平台能正常解析
  • Markdown All in One 更棒的快捷键（比如 Ctrl+M ）和缩进、高亮
  • Vim 啊这个其实和写文章并没有太大关系……

写的时候发现插件商店里的“存储库”链接不见了，害得我每个点每个点。
