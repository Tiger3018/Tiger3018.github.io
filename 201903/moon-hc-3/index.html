美味度

给定长度为 2n2^n2n 的整数排列，其范围 [0,2n)[0, 2^n)[0,2n) 。任意询问某一个区间，希望在至多一次对排列的交换后使得区间异或和，是任意区间最大异或和。求满足条件的区间个数。
n≤20n \leq 20n≤20
据说都是原题可是没找到QAQ

在考场上面看着有序序列的答案和不满足条件的区间，猜了一个假结论。如果直接跑这个假结论就死惨了。

（如果仔细想一下），满足条件即要使得区间异或和等于 2n−12^n-12n−1 ，而异或和无法通过一次交换得到此值的情况当且仅当该区间恰好包括偶数个配对。什么是配对？就是指 ai⊗aj=2n−1a_i \otimes a_j = 2^n-1ai​⊗aj​=2n−1 。

证明：交换一次之后，如果对于该区间内任意一个数 aaa ，满足 sum⊗a⊗b=2n−1sum \otimes a \otimes b = 2^n-1sum⊗a⊗b=2n−1 的 bbb 都是该区间内的数，则该区间在交换后肯定不满足条件。也即 ∑cnt∤2a⊗b=2n−1, a,b∈seq\sum^{cnt\nmid2} a \otimes b = 2^n-1,~a,b\in seq∑cnt∤2a⊗b=2n−1, a,b∈seq （奇数组 a,ba,ba,b ）或 ∑cnt∣2a⊗b=0, a,b∈seq\sum^{cnt\mid2} a \otimes b = 0,~a,b\in seq∑cnt∣2a⊗b=0, a,b∈seq （偶数组 a,ba,ba,b ）

显然奇数组不用交换即可满足条件，偶数组就是偶数配对的条件。这时候最裸的 n4n^4n4 暴力即可转化为确定左端点，遍历维护配对组数就松到 n2n^2n2 。（然而我考试滞留在 n4n^4n4 ）

线性复杂度的处理，需要考虑求不合法区间的值，而通过数据结构将合法区间一个个除去。

魔力树

给定一颗有根树，点上带权值，请你用尽可能少的花费，使得各点的子节点的子树权值和相等。各点修改权值的花费是权值变化的绝对值。
n≤105n \leq 10^5n≤105 且答案需精确到 10−610^{-6}10−6

权值修改可以不是整数……这种做法并不常见。

但是有个基本的点没有变：只修改叶子节点，或者是将对非叶子的修改传下来，一定不会使答案更劣（感性证明）

这时候只需二分整个树的权值和，而我们就可以根据题解的意思做这件事情：
